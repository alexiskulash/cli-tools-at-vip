# Input and Output

---

## Sending output to a file

---

<FullScreenCode>

```bash
$ ls content > files.txt
```

</FullScreenCode>

---

## Where's my output???

---

<FullScreenCode>

```bash
$ cat files.txt
```

</FullScreenCode>

---

<FullScreenCode>

```bash
$ cat files.txt
```

</FullScreenCode>
